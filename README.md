# 🍽️ Sire Restaurant

A modern, responsive restaurant website template built with Next.js, TypeScript, Tailwind CSS, and Sanity CMS. This template provides a complete foundation for a restaurant's online presence, including a dynamic menu, reservations, and content management.

## ✨ Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Dynamic Menu**: Menu items are managed through Sanity CMS for easy updates.
- **Reservation System**: Integrated reservation functionality.
- **Fast Performance**: Built on Next.js with App Router for optimal performance and SEO.
- **Modern Styling**: Styled with Tailwind CSS for rapid UI development.
- **Type Safe**: Fully typed with TypeScript for better developer experience.
- **Google Fonts**: Uses the Geist font family for clean typography.
